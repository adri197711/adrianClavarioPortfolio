# Adrian Clavario Portfolio

Portfolio profesional desarrollado para mostrar mis proyectos, habilidades y experiencia como desarrollador web.

## 🚀 Demo en vivo

🔗 https://adri197711.github.io/adrianClavarioPortfolio/

## 📋 Descripción

Este portfolio fue diseñado para presentar mi perfil profesional de manera moderna, atractiva y completamente responsive. Incluye información sobre mí, mis habilidades técnicas, proyectos destacados y formas de contacto.

## ✨ Características

* Diseño moderno y profesional
* Totalmente responsive (desktop, tablet y móvil)
* Navegación fluida e intuitiva
* Sección de proyectos destacados
* Formulario de contacto
* Optimizado para rendimiento y SEO
* Despliegue automático con GitHub Pages

## 🛠️ Tecnologías utilizadas

* HTML5
* CSS3
* JavaScript (ES6+)
* Git & GitHub
* GitHub Pages

## 📂 Estructura del proyecto

```bash
adrianClavarioPortfolio/
├── index.html
├── style.css
├── script.js
├── assets/
│   ├── images/
│   └── icons/
└── README.md
```

## ⚙️ Instalación y uso local

1. Clonar el repositorio:

```bash
git clone https://github.com/adri197711/adrianClavarioPortfolio.git
```

2. Entrar en la carpeta del proyecto:

```bash
cd adrianClavarioPortfolio
```

3. Abrir `index.html` en tu navegador.

## 🚀 Despliegue

Este proyecto está desplegado en GitHub Pages. Cada cambio enviado a la rama `main` se publica automáticamente.
